**State management:** Replace `ResultsMap` + `numPlayers` with a `CardState[]` array:
```typescript
interface CardState {
    key: number;
    deckId: number | null;
    place: string;
    kills: string;
}
```
- `place` and `kills` are `string` (not number) to let TextField `type="number"` work naturally with empty initial state. Convert to `Number()` on submit.
- Initialize with 2 cards: `[{ key: 0, deckId: null, place: "", kills: "" }, { key: 1, deckId: null, place: "", kills: "" }]`.
- Track `nextKey` state starting at 2, increment on each add.

**Component state:**

**Deck filtering:** Filter decks by selected format (same as current):
```typescript
const filteredDecks = formatID !== 0
    ? data.decks.filter((d) => d.format_id === formatID)
    : data.decks;
```

**Card operations:**
```typescript
const addCard = () => {
    setCards((prev) => [...prev, { key: nextKey, deckId: null, place: "", kills: "" }]);
    setNextKey((k) => k + 1);
};

const removeCard = (key: number) => {
    setCards((prev) => prev.filter((c) => c.key !== key));
};

const updateCard = (key: number, field: keyof Omit<CardState, "key">, value: any) => {
    setCards((prev) => prev.map((c) => c.key === key ? { ...c, [field]: value } : c));
};
```
