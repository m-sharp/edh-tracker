**TabbedLayout.tsx** (per D-04 through D-12):
Create `app/src/components/TabbedLayout.tsx` with the exact implementation from RESEARCH.md Pattern 1:
- Import `ReactElement` from `react`, `useNavigate` and `useSearchParams` from `react-router-dom`, `Box`, `CircularProgress`, `Tab`, `Tabs` from `@mui/material`
- Define `TabConfig` interface: `{ id: string; label: string; content: ReactElement; hidden?: boolean; }`
- Define `TabbedLayoutProps` interface: `{ queryKey: string; tabs: TabConfig[]; loading?: boolean; }`
- Export default function `TabbedLayout({ queryKey, tabs, loading }: TabbedLayoutProps): ReactElement`
- Filter hidden tabs: `const visibleTabs = tabs.filter((t) => !t.hidden);`
- Resolve active tab from query string: `const activeId = searchParams.get(queryKey);` then `const activeIndex = Math.max(0, visibleTabs.findIndex((t) => t.id === activeId));`
- Handle tab change: copy `searchParams` into new `URLSearchParams`, set `queryKey` to `visibleTabs[newIndex].id`, navigate with `replace: true`
- Render `<Tabs value={activeIndex} onChange={handleChange} variant="scrollable" scrollButtons="auto" sx={{ mb: 2 }}>`
- When `loading` is true, render `<Box sx={{ display: "flex", justifyContent: "center", py: 4 }}><CircularProgress /></Box>` instead of tab content
- When not loading, render `activeTab?.content` where `activeTab = visibleTabs[activeIndex]`
